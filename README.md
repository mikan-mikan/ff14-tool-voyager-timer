# FF14 サブマリンボイジャー残り時間表示ツール

FF14のサブマリンボイジャー（潜水艦探索）の帰還時刻を管理するWebツール

🔗 https://ff14-voyager-timer.vercel.app/

---

## ✨ 主な機能

| 機能                     | 説明                                                         |
| :----------------------- | :----------------------------------------------------------- |
| 🚢 4隻の潜水艦を同時管理 | 最大4隻の潜水艦の帰還時刻を一画面で確認                      |
| ⏱️ 2つの入力方式         | 残り時間（例: 2日5時間30分）/ 帰還予定時刻（カレンダー選択） |
| 🔄 リアルタイム更新      | 1秒ごとに自動更新されるカウントダウン                        |
| 🔗 URL共有機能           | 設定をURLに保存し、簡単に共有・ブックマーク可能              |
| 🌓 ダークモード対応      | ライト/ダークモード切り替え可能（設定は自動保存）            |

## 📝 使い方

1. 各潜水艦の帰還時刻を設定（残り時間 or 帰還予定時刻）
2. 「残り時間を表示する」をクリック
3. リアルタイムでカウントダウンを確認
4. 「URLをコピーする」で設定を保存・共有

## 💻 開発者向け

プロジェクトのルートディレクトリで以下のコマンドを実行

| コマンド           | 説明                                    |
| :----------------- | :-------------------------------------- |
| `pnpm install`     | 依存パッケージのインストール            |
| `pnpm run dev`     | 開発サーバー起動 (`localhost:4321`)     |
| `pnpm run build`   | 本番用ビルド (`./dist/`に出力)          |
| `pnpm run preview` | ビルド結果のプレビュー                  |
| `pnpm format`      | コードフォーマット（ESLint + Prettier） |
