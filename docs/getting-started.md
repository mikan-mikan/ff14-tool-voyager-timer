# 🚀 はじめに

> **最終更新**: 2026年1月30日

FF14 サブマリンボイジャー残り時間表示ツールの概要と、開発環境のセットアップ方法を説明します。

---

## プロジェクト概要

FF14（ファイナルファンタジー14）のサブマリンボイジャー（潜水艦探索）の帰還時刻を管理するWebツールです。ゲームにログインしていない時でも、潜水艦の帰還時刻を確認できます。

### 主な特徴

- 4〜24隻の潜水艦を動的に管理
- 2つの入力方式（残り時間 / 帰還予定時刻）
- リアルタイムカウントダウン表示
- URL共有機能
- ダークモード対応

---

| 項目    | バージョン  |

## クイックスタート

### 1. リポジトリをクローン

```bash
git clone https://github.com/kon-k/ff14-tool-voyager-timer.git
cd ff14-tool-voyager-timer
```

### 2. 依存パッケージをインストール

```bash
pnpm install
```

### 3. 開発サーバーを起動

```bash
pnpm run dev
```

ブラウザで http://localhost:4321 にアクセスすると、開発中のサイトが表示されます。
